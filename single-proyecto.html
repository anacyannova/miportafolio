CAPACITACIONES MASS

#WordPress #Diseño UX #Elearning #Desarrollo Web #CMS

SOBRE ESTE PROYECTO

Capacitaciones MASS, empresa de asesoría de prevención de riesgos, necesitaba una plataforma donde sus trabajadores pudieran acceder a cursos de prevención de riesgos de forma ordenada y segura. El resultado fue un sitio web a medida con sistema de acceso por roles, gestión de trabajadores y contenido de capacitación estructurado según las necesidades de las diferentes sucursales de bencineras con las que Capacitaciones MASS trabaja.

PALETA DE COLOR & PROCESO

#PHP #WordPress #JavaScript #Roles De Usuario #CSS #Github #Certificados Automáticos

DETALLES TÉCNICOS

La plataforma fue desarrollada con código propio sobre WordPress, prescindiendo de constructores visuales para tener control total sobre la estructura y el rendimiento. Se implementó un sistema de roles personalizado que permite a los administradores de cada sucursal gestionar sus trabajadores de forma independiente. La entrega de certificados se automatizó mediante un plugin de envío por correo, complementando el botón de descarga integrado en la misma plataforma. Se priorizó una interfaz simple y directa, pensada para usuarios no técnicos que necesitan operar el sistema sin pasos innecesarios.
